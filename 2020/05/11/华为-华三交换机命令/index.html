华为/华三交换机命令
发表于 2020-05-11| 更新于 2020-05-18
阅读量:|评论数:

进入系统视图

system-view

修改交换机名称为XXXXX

[H3C]sysn XXXXX

查看配置：

查看交换机当前配置

[H3C]display current-configuration dis cu

显示arp信息

[H3C]dis arp

显示mac-address

[H3C]dis mac-address

显示端口信息

[H3C]dis interface

查看端口配置

[H3C]dis interf br

查看vlan

[H3C]dis vlan

查看邻居（可以知道该交换机连着哪些别的设备）

[H3C]dis lldp nei br

查看路由表

[H3C]dis ip routing-table

显示版本信息

[H3C]dis version

查看irf拓扑情况

[H3C]dis irf top

Vlan及接口、接口类型

Vlan相当于虚拟接口，是一个抽象的局域网，不同Vlan之间都是独立的。

Vlan的创建

system-view

vlan XX

批量创建vlan

vlan batch 10 20 30 批量创建 vlan 10 20 30

vlan batch 1 to 3 批量创建vlan 1、2、3

接口

交换机接口一般是eth百兆接口和Gth千兆接口，还有万兆的TENG口。

想要进入这些接口直接在对应接口名称前面加入interface就行了

interface Ethernet0/0/1

interface GigabitEthernet0/0/1

interface TENG0/0/1

进入接口后可以使用一下命令来直接查看当前接口下的配置

dis this

接口类型

Access、Trunk、Hybrid和聚合口Eth-trunk

Access常用于接入交换机与用户终端之间接口的配置，可默认通过某个vlan，在[Huawei]模式下

port link-type access

port default vlan 1

Trunk常用于交换机之间连接配置，可配置多个vlan通过，在[Huawei]模式下

port link-type trunk

port trunk allowed vlan

Hybrid是华为华三交换机专门的一个类型

Eth-trunk聚合口是华为华三的端口聚合，可以吧两个端口作为聚合组来使用。

interface Eth-Trunk1

port link-type trunk

undo port trunk allow-pass vlan 1

port trunk allow-pass vlan 2 to 4094

mode lacp-static

dhcp snooping trusted

路由

静态路由

静态路由意思挺简单，就是自己加路由，目的地址+掩码，下一跳的地址，比如：

Ip route-static 0.0.0.0 0.0.0.0 10.1.1.254

这个意思就是，不管去哪里，下一跳地址都是10.1.1.254

动态路由

RIP

OSPF

堆叠IRF

不同交换机对最大的堆叠数量可能有影响。

我对堆叠的理解是两个或多个交换机成为一个想象中的一个交换机，配置起来也很简单，两个交换机，加到一个堆叠组，然后一个做为主一个作为辅，就像一个小组里面必须要有个组长一样。两台堆叠无需做环状，两台以上可做。

H3C堆叠操作流程：

1、 master交换机改变权限，irf member 1 priority 32

2、进堆叠端口 int Ten-GigabitRthernet 1/1/1，int Ten-GigabitRthernet 1/1/2

3、端口关闭 shutdown

4、开启irf虚拟端口 irf-port 1/1 irf-port 1/2

5、把物理端口划入虚拟端口 port group interface Ten-GigabitEthernet 1/1/1 port group interface Ten-GigabitEthernet 1/1/2

6、进物理端口，打开物理端口 int Ten-GigabitRthernet 1/1/1，int Ten-GigabitRthernet 1/1/2

Undo shutdown

7、irf-prot-configuration active

8、save保存配置

交换机2

1、更换板载（irf member 1 remember 2）

2、其余操作与上述相同

3、记得保存，然后才能插堆叠线，正常配置好后，连接堆叠线2号交换机会自动重启。

dis irf top

VRRP虚拟网关

VRRP感觉跟堆叠挺像的，区别在一个是交换机一个是路由器。两个路由器一个主一个辅，然后就像两个班主任带领各自的班级，1班和二班，但是比如有一天1班老师请假了，那么1班的学生可以找2班班主任请假什么的，这么个形容。

账号登录

local-user XXX （用户名） password simple XXX（密码）

local-user XXX service-type ppp

定义acl策略

acl是交换机的一个安全策略，可以自己设定规则，在规则组内的某些条件能做什么。

定义acl

acl number XXXX（3000以上）进入以后

rule permit/deny IP/TCP/UDP等 sourceXXX.XXX.XXX.XXX XXX.XXX.XXX.XXX（反掩码）

destination XXX.XXX.XXX.XXX XXX.XXX.XXX.XXX（反掩码）eq

注意：华为默认没有deny any any

防火墙上端口加载ACL

firewall packet-filter 3000 inbound
